Trinity

Date : Novembre 2024 - Février 2025

Organisation : EPITECH

Objectif : Développement d'un système complet DevOps, web et mobile pour une chaîne de supermarchés, visant à améliorer l'expérience utilisateur et optimiser la gestion des stocks.

Composition du projet : Le projet est divisé en trois phases :

  • DevOps : Création et mise en place d'une pipeline CI/CD pour automatiser les déploiements et maintenir la qualité du code, incluant des tests unitaires et la configuration de runners GitLab sur une machine virtuelle.
  • DevWeb : Développement d'une application web avec une API RESTful pour la gestion des stocks et des ventes, permettant aux managers de visualiser des indicateurs de performance et de gérer les clients et les produits.
    Technologies : Django pour le backend, React pour le frontend.
  • DevApp : Conception d'une application mobile intuitive pour simplifier le processus d'achat des utilisateurs, avec un scanner de produits, un panier et une intégration de paiement via PayPal.

Livrables : Code source, documentation technique et diagrammes UML, pipeline CI/CD fonctionnel avec tests unitaires, et application mobile déployable sur un appareil physique (Android ou iOS).

TimeManager

Date : Octobre 2024

Organisation : EPITECH

Objectif : Développement d'une API multiplateforme pour la gestion du temps, incluant les fonctionnalités d'authentification et de visualisation des données.

Missions : Création d'une API permettant la gestion des utilisateurs, des horaires de travail et des pointages. Développement d'une interface de visualisation en Vue.js pour l'analyse des données de travail, déploiement avec Docker et intégration continue pour assurer la rapidité, la sécurité et la fiabilité du système.

Technologies utilisées : Node.js, Express, Vue.js, Docker, CI/CD, JWT pour l'authentification, et base de données SQL.
