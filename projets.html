Mes Projets

Trinity

Date : Novembre 2024 - Février 2025

Organisation : EPITECH

Description : Développement d’un système DevOps, web et mobile pour une chaîne de supermarchés.

Principaux défis : Coordination entre plusieurs équipes (mobile, web, et DevOps), et gestion des tests automatisés pour un grand volume de données.

Technologies clés : Django, React, Docker, Jenkins (CI/CD).

Impact : Une meilleure gestion des stocks, des paiements simplifiés pour les clients via mobile, et une réduction de 30 % des erreurs dans la logistique.

JACK THE RIPEUR - Système de Gestion des Déchets 2.0

Date : Novembre 2024 - Février 2025

Description : Création d’une solution intelligente pour optimiser la gestion des déchets en ville.

Fonctionnalités clés : Bacs connectés, signalements citoyens via une application, et optimisation des itinéraires pour les camions grâce à des algorithmes de routage.

Technologies utilisées : Flutter pour l'application mobile, API REST en Node.js, et capteurs IoT pour les bacs.

Résultat : Une amélioration de 40 % dans la gestion des trajets des camions et une réduction des plaintes citoyennes.

TimeManager

Date : Octobre 2024

Objectif : Développer une API multiplateforme pour la gestion du temps et des horaires.

Missions : Conception de l'API backend avec Node.js et Express, mise en place de Docker pour le déploiement, et intégration CI/CD via GitHub Actions.

Technologies utilisées : Node.js, Vue.js, JWT, Docker, PostgreSQL.

Impact : Automatisation des tâches administratives liées à la gestion du temps, et gain de 25 % d'efficacité pour les utilisateurs finaux.
